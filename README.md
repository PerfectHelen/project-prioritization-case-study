# 📊 Project Prioritization Case Study

## 📌 Overview

This project demonstrates strategic decision-making under financial constraints.
The goal was to select the most viable project from three options using cost-benefit and ROI analysis.

---

## ❗ Problem Statement

An organization had limited funds and could only invest in **one of three projects**:

* Upgrading Office Technology
* Expanding Customer Service Training
* Launching a Small Marketing Campaign

---

## 🧠 Approach

Each option was evaluated based on:

* Cost implications
* Expected outcomes
* Advantages and disadvantages
* Return on Investment (ROI)

---

## 📊 Final Recommendation

✔ **Small Marketing Campaign**

### Why?

* Low implementation cost
* Fast measurable results
* High potential ROI
* Ability to fund future projects

---

## 💡 Key Insight

> Strategy is not about doing everything.
> It is about doing the right thing first.
